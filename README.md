# Appinapp

Sistema di gestione immobili in Appennino con vetrina pubblica e gestione multi-utente.

## Descrizione

Appinapp è una piattaforma web moderna per la gestione e pubblicazione di immobili. Il sistema permette a proprietari e agenzie di pubblicare i propri appartamenti e case, gestendo tutto attraverso un'interfaccia intuitiva con diversi livelli di accesso.

## Tecnologie Utilizzate

- **Frontend**: React 18 + Vite
- **Backend**: Supabase (PostgreSQL)
- **Deploy**: Vercel
- **Autenticazione**: Supabase Auth
- **Styling**: CSS3 con animazioni moderne

## Funzionalità Principali

### Sistema Multi-Utente
- **Admin**: Controllo completo del sistema, gestione utenti e approvazione immobili
- **Editor**: Inserimento e modifica dei propri immobili
- **Viewer**: Visualizzazione e consultazione immobili

### Gestione Immobili
- Inserimento completo dati immobile (titolo, descrizione, prezzo, caratteristiche)
- Upload multiplo immagini con anteprima
- Geolocalizzazione con coordinate
- Categorie: appartamento, villa, attico, monolocale, bilocale, trilocale
- Stati: disponibile, affittato, venduto, in trattativa

### Vetrina Pubblica
- Visualizzazione immobili pubblicati
- Sistema di ricerca e filtri avanzati
- Galleria immagini responsive
- Sistema preferiti per utenti registrati
- Form contatto per richieste informazioni

### Dashboard Amministrativa
- Statistiche visualizzazioni
- Gestione richieste contatto
- Moderazione contenuti
- Gestione utenti e permessi
